GIT COMMAND

  • git add -чтобы включить изменения файлов в нашу следующую фиксацию.

  • git status - Команда Git status предоставляет нам всю необходимую информацию о текущей ветке

  • git push - отправить изменения на удаленный сервер

  • git commit - сохранить наши изменения

  • git diff - Эта команда показывает различия в файлах, которые еще не были инсценированы

  • git rm - Эта команда удаляет файл из рабочего каталога и завершает удаление.

  • git clone - создает идентичную копию последней версии проекта в репозитории и сохраняет ее на вашем компьютере.
